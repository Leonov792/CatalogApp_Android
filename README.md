# ?? Product Catalog — Android App

E-commerce style product catalog with search and cards.

## ?? Tech Stack
- **Kotlin** + Coroutines
- **MVVM** Architecture
- **LiveData** for reactive UI
- **RecyclerView** with CardView
- **Glide** for image loading
- **SearchView** for filtering

## ?? How to Build
1. Open in **Android Studio** (Hedgehog or newer)  
2. Sync Gradle  
3. **Build > Build APK**

Or via CLI: ./gradlew assembleDebug
